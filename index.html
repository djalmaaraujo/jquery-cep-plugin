Example 2: Event Change in non-multiple selects

This example will do a console.log with the response from webservice. Enable you debug console (like firebug or chrome console..).

Some infos

  • JSON response data;
  • WebService used: http://apps.pianolab.com.br/wscep/ (No database stored. This webservice is just a wrapper that grabs informations from http://republicavirtual.com.br/cep/). No CEP database is used on pianolab's webservice.
  • Callback options: callback: function (data) {} or ".some-classe" (.jcep-target is a default classe)
  • Json template: (You can change passing on options if you want to change the webservice)
    json_template: {
		result: 'resultado',
		result_message: 'resultado_txt',
		state: 'uf',
		city: 'cidade',
		neighborhood: 'bairro',
		street_type: 'tipo_logradouro',
		street: 'logradouro'
	}
